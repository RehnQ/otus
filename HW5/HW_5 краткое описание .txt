База с электронными курсами


В базе присутствуют таблицы:

Students, с дополнительными параметрами (дата рождения) + информация из таблицы positions для сбора статистики
Courses, c наименованиями, статусом (активный, в разработке, архивный), стоимостью, максимальным и проходным баллом
Active Learnings (для отслеживания прогресса студентов по актуальному обучению)
Learnings c датами начала и окончания обучения и результатом
Accounts со счетами студентов
Transactions для отслеживания прихода/расхода средств

Status_course - словарь для статусов курсов
Status_learning - словарь для статусов по обучению
